# LLM Engineering

A hands-on repository for building and exploring LLM applications, retrieval workflows, vector search, and practical AI experiments through notebooks and supporting data assets.

## Overview

This project is a notebook-driven workspace for learning and experimenting with:

- Large Language Model (LLM) workflows
- Retrieval-Augmented Generation (RAG) concepts
- Vector databases and retrieval pipelines
- Data preprocessing for AI applications
- Evaluation and testing of LLM-driven workflows

The repository is organized as a practical learning and experimentation environment, with step-by-step notebooks, data assets, and retrieval resources.
